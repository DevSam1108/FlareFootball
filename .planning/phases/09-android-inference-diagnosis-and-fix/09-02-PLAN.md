## DIAG-04: Root Cause Identified and Fixed

**Root cause:** Missing `aaptOptions { noCompress 'tflite' }` in `android/app/build.gradle`.

**Mechanism:** Android's AAPT compresses all assets by default. A compressed `.tflite` file cannot be memory-mapped by `FileUtil.loadMappedFile()`. The interpreter constructor throws (silently from Flutter's perspective; visible in logcat under `ObjectDetector` tag). With a null interpreter, `ObjectDetector.predict()` returns empty results. The `predictor?.let` guard in `YOLOView.onFrame()` short-circuits. `streamCallback` is never invoked. The EventChannel delivers no data. `onResult` never fires.

**Fix applied:** Added `aaptOptions { noCompress 'tflite' }` inside the `android {}` block in `android/app/build.gradle` (Plan 01, Task 1).
