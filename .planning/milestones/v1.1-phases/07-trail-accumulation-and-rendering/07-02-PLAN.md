Create `lib/screens/live_object_detection/widgets/trail_overlay.dart` containing a `TrailOverlay` class extending `CustomPainter`.

**Imports:**
- `package:flutter/material.dart` (for `CustomPainter`, `Canvas`, `Size`, `Paint`, `Colors`, `Offset`, `PaintingStyle`, `StrokeCap`)
- `package:tensorflow_demo/models/tracked_position.dart`
- `package:tensorflow_demo/utils/yolo_coord_utils.dart`

**Constructor fields:**
- `required List<TrackedPosition> trail` — the position history from BallTracker
- `required Duration trailWindow` — matches BallTracker's window for age calculation
- `double cameraAspectRatio = 16.0 / 9.0` — same default as DebugDotPainter

**`paint(Canvas canvas, Size size)` method:**

1. If `trail.isEmpty` or `size.isEmpty`, return immediately.
2. Capture `final now = DateTime.now()` and `final windowMs = trailWindow.inMilliseconds.toDouble()`.

3. **Draw connecting line segments (RNDR-02 + RNDR-03):**
   - Create a `Paint` with `strokeWidth: 2.5`, `strokeCap: StrokeCap.round`, `style: PaintingStyle.stroke`.
   - Loop from `i = 1` to `trail.length - 1`:
     - Get `prev = trail[i - 1]` and `curr = trail[i]`.
     - **RNDR-03:** If `prev.isOccluded || curr.isOccluded`, `continue` (skip gap).
     - Compute age: `now.difference(curr.timestamp).inMilliseconds / windowMs`.
     - Compute opacity: `(1.0 - age).clamp(0.0, 1.0)`.
     - Set line color: `Colors.orange.withValues(alpha: opacity * 0.7)`.
     - Draw line: `canvas.drawLine(YoloCoordUtils.toCanvasPixel(prev.normalizedCenter, size, cameraAspectRatio), YoloCoordUtils.toCanvasPixel(curr.normalizedCenter, size, cameraAspectRatio), linePaint)`.

4. **Draw dots at each non-occluded position (RNDR-01):**
   - Create a `Paint` with `style: PaintingStyle.fill`.
   - For each `pos` in `trail`:
     - If `pos.isOccluded`, `continue`.
     - Compute age: `now.difference(pos.timestamp).inMilliseconds / windowMs`.
     - Compute opacity: `(1.0 - age).clamp(0.0, 1.0)`.
     - Compute radius: `5.0 * opacity + 2.0` (tapers with age — newest dots are ~7px, oldest ~2px).
     - Set dot color: `Colors.orange.withValues(alpha: opacity)`.
     - Draw circle: `canvas.drawCircle(YoloCoordUtils.toCanvasPixel(pos.normalizedCenter, size, cameraAspectRatio), radius, dotPaint)`.

**CRITICAL (RNDR-05):** All coordinate mapping MUST use `YoloCoordUtils.toCanvasPixel()` — the shared utility created in Plan 01. Do NOT inline the crop math. Do NOT use simple `normalized.dx * size.width` — that ignores the FILL_CENTER crop and will produce the same Y-offset bug fixed in Phase 6.

**`shouldRepaint(TrailOverlay old)` method:**
Return `true` always. Rationale from research: `BallTracker.trail` returns `List.unmodifiable()` which creates a new wrapper each call, so reference equality is unreliable. The real performance guard is `RepaintBoundary` wrapping this painter, not `shouldRepaint`. Since `setState` only fires on detection callbacks (not every vsync frame), `paint()` is called at detection frame rate which is correct.

```dart
YOLOResult? _pickBestBallYolo(List<YOLOResult> results) {
  // TRAK-03: Reject tennis-ball; accept Soccer ball and ball only.
  const priority = {'Soccer ball': 0, 'ball': 1};
  final candidates = results
      .where((r) => priority.containsKey(r.className))
      .toList();
  if (candidates.isEmpty) return null;

  // Sort by class priority first, then confidence within same class.
  candidates.sort((a, b) {
    final pa = priority[a.className]!;
    final pb = priority[b.className]!;
    if (pa != pb) return pa.compareTo(pb);
    return b.confidence.compareTo(a.confidence);
  });

  // TRAK-04: If multiple candidates share the top priority,
  // use nearest-to-last-known-position as tiebreaker.
  final topPriority = priority[candidates.first.className]!;
  final topCandidates = candidates
      .where((r) => priority[r.className] == topPriority)
      .toList();

  if (topCandidates.length == 1) return topCandidates.first;

  final lastKnown = _tracker.lastKnownPosition;
  if (lastKnown == null) return topCandidates.first; // no history

  // Pick candidate whose normalizedBox center is closest to last known.
  topCandidates.sort((a, b) {
    final da = _squaredDist(a.normalizedBox.center, lastKnown);
    final db = _squaredDist(b.normalizedBox.center, lastKnown);
    return da.compareTo(db);
  });
  return topCandidates.first;
}

/// Squared Euclidean distance — no sqrt needed for comparison.
double _squaredDist(Offset a, Offset b) {
  final dx = a.dx - b.dx;
  final dy = a.dy - b.dy;
  return dx * dx + dy * dy;
}
```
