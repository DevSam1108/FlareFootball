**OVLY-02:** Extend the SSD detection path in `live_object_detection_screen.dart` to also update `_debugDotPosition`.

**Step 1: Add `_pickBestBallSsd` helper method**

Add to `_LiveObjectDetectionScreenState`:
```dart
Offset? _pickBestBallSsd(List<DetectedObjectDm> objects) {
  final previewSize = ScreenParams.screenPreviewSize;
  if (previewSize.width == 0 || previewSize.height == 0) return null; // guard against zero-size
  // SSD MobileNet uses COCO labels — 'sports ball' is the relevant class
  final candidates = objects.where((o) => o.label.toLowerCase().contains('ball')).toList();
  if (candidates.isEmpty) return null;
  candidates.sort((a, b) => b.score.compareTo(a.score));
  final best = candidates.first;
  // renderLocation is in screen-pixel space; normalize to 0.0-1.0
  return Offset(
    best.renderLocation.center.dx / previewSize.width,
    best.renderLocation.center.dy / previewSize.height,
  );
}
```

The zero-size guard prevents division by zero if `ScreenParams.screenPreviewSize` has not been populated yet (defensive programming per research pitfall #3).
